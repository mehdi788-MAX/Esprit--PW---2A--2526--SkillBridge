# SkillBridge – Plateforme Web

## Overview
This project was developed as part of the Web Technologies Project (2A) at **Esprit School of Engineering – Tunisia** (Academic Year 2025–2026).

SkillBridge is a web application that allows users to communicate, manage profiles, and interact through a messaging system.

## Features
- User authentication
- Profile management
- Messaging system
- Search conversations
